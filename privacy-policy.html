개인정보처리방침

시행일자: 2024년 1월 1일

버스 도착 알림 앱(이하 "앱")은 사용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 개인정보처리방침은 앱이 어떤 정보를 수집하고, 어떻게 사용하며, 어떻게 보호하는지 설명합니다.

1. 수집하는 개인정보

앱은 본질적인 서비스 제공을 위해 필요한 최소한의 정보만 수집합니다.

  • 위치 정보 (선택): 근처 정류소를 찾기 위해 GPS 좌표를 일시적으로 사용합니다. 위치 정보는 앱이 실행 중일 때만 처리되며, 서버에 저장되지 않습니다.
  • 즐겨찾기 정보 (선택): 사용자가 저장한 정류소 즐겨찾기를 기기 내에만 저장합니다.
  • 푸시 알림 토큰 (선택): 알림 제공을 위해 기기 토큰을 수집할 수 있습니다.

앱은 다음 정보를 자동으로 수집할 수 있습니다: 기기 종류, OS 버전, 앱 버전, 서비스 이용 기록(버그 보고, 오류 로그). 이러한 정보는 개인을 식별할 수 없는 형태로 수집됩니다.

2. 수집한 정보의 사용 목적

  • 실시간 버스 도착 정보 제공
  • 사용자 맞춤 정류소 추천
  • 푸시 알림 서비스 제공
  • 앱 서비스 품질 개선 및 버그 수정
  • 법령상 의무 이행

3. 개인정보의 제3자 제공

앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 아래의 경우 예외로 합니다:

  • 사용자의 별도 동의를 받은 경우
  • 법령의 규정에 의한 경우 (수사기관의 요청 등)
  • 버스 정보 제공을 위한 외부 API 호출 (서울시 버스 위치 정보)

4. 외부 API 사용

앱은 실시간 버스 도착 정보를 위해 서울시 열린데이터광장 버스 도착 정보 API를 사용합니다. 해당 API로의 요청에는 버스 정류소 검색 정보가 포함될 수 있으며, 각 데이터 제공자의 개인정보처리방침이 적용됩니다.

5. 개인정보의 보관 기간

  • 위치 정보: 앱 종료 시 즉시 삭제
  • 즐겨찾기 정보: 사용자가 삭제할 때까지 (기기 저장)
  • 푸시 토큰: 서비스 탈퇴 시 또는 삭제 요청 시까지
  • 이용 기록: 익명 통계 데이터로 1년간 보관 후 삭제

6. 사용자의 권리

사용자는 언제든지 다음과 같은 권리를 행사할 수 있습니다:

  • 개인정보 열람 및 정정 요구
  • 개인정보 삭제 요구
  • 개인정보 처리 정지 요구

앱 내 설정 메뉴에서 직접 즐겨찾기 정보를 삭제할 수 있으며,其他的 개인정보 삭제 요청은 하단 연락처로 문의주시기 바랍니다.

7. 쿠키 및 유사 기술

앱은 쿠키를 사용하지 않습니다. 다만, 서비스 분석을 위해 익명화된 사용 통계 데이터를 수집할 수 있습니다.

8. 개인정보의 안전성 확보

앱은 수집된 개인정보를 안전하게 보호하기 위해 다음과 같은 조치를 취합니다:

  • 개인정보는 암호화된 통신(HTTPS)을 통해 전송
  • 필요한 최소한의 정보만 수집
  • 정기적인 보안 업데이트

9. 만 14세 미만 아동

앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만 아동이 서비스를 사용하고 있음을 발견한 경우,监护인의 동의 없이 해당 정보가 수집되지 않도록 조치를 취합니다.

10. 개인정보처리방침 변경

본 개인정보처리방침은 법령, 정책 또는 서비스 변경에 따라 수정될 수 있습니다. 중요한 변경이 있는 경우 앱 내 공지 또는 별도 통지를 통해 사전 안내드립니다.
